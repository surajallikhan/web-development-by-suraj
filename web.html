Definition

A Library Management System (LMS) is a software application that helps in managing library resources, including books, users, borrowing transactions, and fines. This research focuses on comparing the implementation of an LMS using Python with JSON storage versus Python with NetworkX graph-based representation to analyze performance, efficiency, scalability, and ease of development. Sen, S., Singh, N., & Singh, A. (2012). Library Management System.

Why is it important in todayâ€™s world?

As libraries transition to digital platforms, the choice of data storage and processing plays a significant role in managing large datasets effectively. JSON-based storage provides a lightweight, document-based approach, whereas NetworkX leverages graph structures to model relationships between books, users, and transactions. This study helps determine the best approach for handling real-world LMS applications in Python.

Applications of the research

The findings can help developers and institutions select the best approach for implementing digital library systems. The performance analysis can guide developers in choosing graph-based or document-based storage for large-scale data-driven applications. Insights from this research can also be applied to other management systems, such as student records, hospital databases, or social network analysis.
